This plugins provides the functionality to serve native ads on your blog.

== Description ==

This plugin is great for publishers which are using this CMS, the native.ad plugin can add the integration via "Sidebar" by simply installing it.

With the plugin you can also integrate automatically with our ad serving technology by including the javascript tag on the header of your site. You can do it with just one click on the admin menu of the plugin.


== Screenshots ==

1. Search for our plugin directly on your wordpress.
2. Easy to setup.
3. Include the widget on your sidebar area.


== Installation ==

- Go to the plugin section on Wordpress
- Click on the click at the top "add new"
- Search for "NativeAD"
- Cick on "Install now"
